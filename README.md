# 🎓 SchoolProject - QR-код система посещаемости

Простая и быстрая система регистрации учеников с персональными QR-кодами для отслеживания посещаемости.

## 🚀 Быстрый старт

### 1️⃣ Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2️⃣ Запуск сервера
```bash
python3 app.py
```

Откройся в браузере: **http://localhost:5000**

## ✨ Основные функции

### 📝 **Регистрация**
- Введи данные: имя, email, телефон, класс
- Автоматически генерируется уникальный QR-код
- QR-код отображается на экране
- Данные сохраняются в БД

### 📱 **Сканирование QR-кода**
- Использует камеру для сканирования
- Или ввод ID вручную
- При сканировании записывается в БД время сканирования
- Выводится информация об ученике

### 📊 **База данных**
- Таблица **Student** - информация об учениках
- Таблица **QRScan** - история всех сканирований

## 🔌 API Endpoints

### Регистрация ученика
```bash
POST /api/register
```
**Параметры:**
```json
{
  "fullname": "Иван Петров",
  "email": "ivan@example.com",
  "phone": "+7 999 123-45-67",
  "grade": "10А"
}
```

### Сканирование QR-кода
```bash
POST /api/scan-qr
```
**Параметры:**
```json
{
  "student_id": "uuid-string"
}
```

### Получить ученика
```bash
GET /api/student/<student_id>
```

### История сканирований ученика
```bash
GET /api/scans/<student_id>
```

### Все ученики
```bash
GET /api/students
```

### Статистика
```bash
GET /api/stats
```

## 🗄️ База данных

### Таблица Student
| Поле | Тип | Описание |
|------|-----|----------|
| id | Integer | Уникальный ID |
| student_id | String(36) | UUID ученика (в QR-коде) |
| fullname | String(100) | Полное имя |
| email | String(100) | Email |
| phone | String(20) | Номер телефона |
| grade | String(10) | Класс (9А, 10Б и т.д.) |
| qr_code | Text | QR-код в формате base64 |
| created_at | DateTime | Дата регистрации |

### Таблица QRScan
| Поле | Тип | Описание |
|------|-----|----------|
| id | Integer | Уникальный ID |
| student_id | String(36) | ID отсканировавшего ученика |
| scanned_at | DateTime | Время сканирования |

## 📋 Как использовать

### Для ученика (регистрация):
1. Открой http://localhost:5000
2. Нажми "📝 Зарегистрироваться"
3. Заполни форму
4. Получи свой QR-код
5. Сохрани или сфотографируй код

### Для проверки посещаемости (администратор):
1. Открой http://localhost:5000
2. Нажми "📱 Сканировать QR-код"
3. Наведи камеру на QR-код ученика
4. Система автоматически зафиксирует посещение
5. Информация сохранится в БД

## 🛠️ Технологический стек

- **Backend:** Python + Flask
- **БД:** SQLite
- **Frontend:** HTML5 + CSS3 + JavaScript
- **QR-код:** qrcode + Pillow
- **Сканирование:** jsQR.js

## ⚠️ Примечания

- БД создаётся автоматически при первом запуске
- QR-коды генерируются с uuid4 для уникальности
- Сканирование работает на любом устройстве с камерой
- Все данные хранятся локально в SQLite
